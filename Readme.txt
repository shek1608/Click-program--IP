click_ex1.click contains the code for the question.
click_ex1_counters.dump contains the count of all packet types.
click_ex1_udp.dump contains a dump of the UDP packets that were present.

To run the .click file, 
-> Copy the 'click_ex1.click' file as well as the 'click_ex1.dump' into the 'userlevel' folder of click.
-> In terminal, go to the userlevel folder and run: ./click click_ex1.click and the output files will be created.
